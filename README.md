<h2>5. 느낀점</h2>
<h3>5-1. 어려웠던 부분</h3>
📌request 영역과 param 영역에 대한 어려움<br>
✔ 데이터를 전달하는 방식에 대해 각 영역에 대한 이해가 부족하여 기능 구현에 어려움을 겪었습니다.<br>
forward, redirect, 그리고 queryString의 차이를 이해하고, 상황에 맞는 영역을 적절히 활용할 수 있는 역량을 갖추게 되었습니다.<br>
<br>
📌Git에 대한 어려움<br>
✔ 협업을 위해 처음 Git을 사용하면서, Git의 설정과 다양한 기능들을 익히는 데 어려움이 있었습니다.<br>
지금은 팀장과 팀원 역할을 모두 수행할 수 있는 역량을 갖추게 되었습니다.
<br>
<h3>5-2. 총평</h3>
<strong>📝결국은 JAVA다. </strong><br><br>
💊JSP 설계 방식에 대해서는 잘 이해했지만, 기능 구현은 여전히 어려웠습니다. 그 이유는 바로 Java였습니다.<br>
이번 프로젝트를 통해, Java라는 언어가 개발에 얼마나 중요한지 깨달을 수 있었습니다.<br>
Java의 기본 개념, 컬렉션 프레임워크 등 개발에 필요한 많은 지식이 결국 Java에서 비롯된다는 사실을 확실히 알게 되었습니다.<br>
개발자로서 나아가기 위해서는 Java에 대한 깊은 이해가 뒷받침되지 않으면 쉽게 발전할 수 없다는 것을 느꼈습니다.<br>
앞으로 새로운 기술과 지식을 습득하는 과정 속에서도 Java 공부는 꾸준히 병행할 것입니다.🔔
